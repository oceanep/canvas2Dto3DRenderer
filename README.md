# raw2dto3d

This is just a test project to recreate 3D rendering effects using the canvas 2D renderer. I will add functionality to it over time.

## Functionality available

- Vertex rendering
- Face rendering
- **Rotation**
  - X-Axis
  - Y-Axis
  - Z-Axis
- **Translation**
  - X-Axis
  - Y-Axis
  - Z-Axis
